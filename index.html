Interactive Maps for the browser.
Easily create interactive maps with Tyler, an extremely lightweight JavaScript library with zero dependencies and crafted in under 300 lines of code.
Try it out:
Simple and Lightweight
Tyler is designed to be incredibly easy to use and maintain, with less than 300 lines of code. There are no external dependencies, making it a hassle-free choice for adding interactive maps to your projects.
OpenStreetMap Integration
By default, Tyler uses OpenStreetMap tiles for rendering your maps. This provides a reliable and free source of map data for your projects.
Usage
You can include it in your project using one of the following methods.
From a package manager
You can add Tyler to your project using a package manager like NPM or Yarn: 
## Install using npm:
$ npm install --save tyler-js

## Install using yarn:
$ yarn add tyler-js
